# Solana Frontier Hackathon

The best Solana hackathon projects start with a real problem and a small, working demo. This repo collects ideas and resources to help you get there faster on Solana.

## How to Use this Repository

### Using AI ([![Ask DeepWiki](https://deepwiki.com/badge.svg)](https://deepwiki.com/Lightprotocol/frontier-hack))

This will show you relevant resources for your application.

1. Open [deepwiki.com/Lightprotocol/frontier-hack](https://deepwiki.com/Lightprotocol/frontier-hack)
2. Paste Your Idea and additional context in the chat bar to "AskDevin"
3. Devin will give you the relevant resources and AI skills for your hackathon project.

> You can use [DeepWiki](https://deepwiki.com/) to index any public GitHub repository.
